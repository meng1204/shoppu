The Application such as static pages (homepage) and order are requested to use the layout
from app/views/layouts and rendered the head, body, footer html.erb



look at the html tag and see the class of each element. Each style
of those classes are in the boostrap.css.

The usage of the class is about how bootstrap style each element.

Any styling of header and footer are in the application.css




Styling on the front page can be found in the static_page.scss. 


Styling of the user profile page can be found on the user.SCSS.
